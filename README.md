# 马队积分（MaDui Score）

一个为多人扑克「马队」玩法设计的 H5 计分小工具。手机端浏览器优先，所有数据本地存储，无需后端、无需联网、无需登录。

## 功能特性

- **对局设置**：可选 2 / 3 / 4 队、每队人数、扑克牌副数（1~5 副）、出锅上限、自定义队名
- **本轮录入**：每轮按队伍录入 delta 分数，自动校验：
  - 必须是 5 的倍数，且在 `[-人数*20, 牌面分+人数*20]` 区间
  - 各队 delta 之和必须等于本轮总牌面分（牌副数 × 100）
- **撤销**：以「整轮提交」为单位回滚，分数和大剔标记同步还原
- **大剔（一方完胜）**：
  - 仅 2 队对局生效
  - 判定基于**本轮 delta**：一方 ≥ 牌面总分（含倒扣）且另一方为负分
  - 触发后**不结束本局**，仅播放全屏庆祝动效，可在同一局内多次触发
  - 战绩中标记被大剔的队伍
- **结算**：任一队伍累计 ≥ 出锅上限自动结算，跳转结算页（撒花、获胜队伍金色高亮、得分明细）
- **战绩回看**：胜场统计、回合记录，标识每局是否触发过大剔，可点击徽章回放大剔动效
- **本地持久化**：设置、当前对局、历史战绩均存于 localStorage，刷新不丢
- **轻量动效开关**：低端机可开启「精简庆祝动效」减少粒子数
- **防误触**：返回开局设置、清空当局、撤销均有二次确认弹窗

## 技术栈

- Vue 3.5 + `<script setup>` + Composition API
- TypeScript（strict）
- Vite 5
- 纯 CSS（无 UI 库）

## 项目用法

### 环境

- Node.js ≥ 18

### 安装与运行

```bash
# 安装依赖
npm install

# 本地开发（默认 http://localhost:5173）
npm run dev

# 生产构建（含 vue-tsc 类型检查）
npm run build

# 预览生产产物
npm run preview
```

### 部署

`npm run build` 产物在 `dist/`，纯静态文件，可直接放任意静态托管（Nginx、对象存储、Vercel、GitHub Pages 等）。

## 使用流程

1. **开局设置**：选择队伍数、每队人数、牌副数、出锅上限，自定义队名
2. **开始游戏** → 进入计分页
3. 每打完一手牌，按队伍录入本轮分数（正负均可），合计需等于牌面总分，点「确认」
4. 触发大剔时自动播放动效，本局继续
5. 任一队累计 ≥ 出锅上限，自动跳转结算页
6. 「再来一局」开始下一轮，「查看战绩」回看历史

## 数据存储

- key：`maduiScore.v1.4`
- 数据结构变更时会升级 key 失效旧缓存
- 清空浏览器站点数据即可彻底重置
